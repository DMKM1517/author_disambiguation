# Web Application for Author Disambiguation

To run:

 - Install NodeJS
 - In the `web` directory run:
	 - `npm install`
 - Create a config file `web/config.json` with the following template:
```
{
	"clientId": ****,
	"clientSecret": "<secret>",
	"localhostUrl": "localhost"
}
```
 - Run the application
	 - `node .`
